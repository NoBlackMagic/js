Variables JS Spells | Basic

Variables are containers of data: numbers, strings, ...

You can think to a variable as a box with a label.

It doesn't really matter what you put inside that box, the important part is that you can refer to the box content using its label.

Create a brand new variable

To create a new variable you can use the operator var followed by the name you want to assing to that variable: 

var name;

This is a complete and valid instruction which means: 

» make a box and apply the label "name" on it

Feed variables with contents

                

                    Empty boxes are just useless so we need to learn how to
                    put some content into a variable.
                

                

                    The simplest way to do it is when you create the new variable:
                

                

                    
var name = 'Luke Skywalker';

                

                

                    But you can also use existing variables and change their contents on the fly:
                

                
name = 'Darth Vader';

                

                    
                        NOTE: if you have already created the variable then is an error to 
                        use the var operator again! It's like destroying the box and then
                        create a new one with the same lable!

Reading contents fro a variable

                

                    When a variable exists you can access it's content by just using the variable name:
                

                
var name = 'Luke Skywalker';
console.log(name); // -> Luke Skywalker

                

                   When it comes to read variable it is more likely that you want to perform some operations
                   within variables and - maybe - save the result in a brand new one:
                

                
var num1 = 10;
var num2 = 20;
var result = num1 + num2;

                

                    In the last line of the previous code we read num1 and num2 values, we use
                    them within a sum operation and then we assign the resulting value to a new
                    variable named result.
                

                

                    It is actually pretty important to understand that in this single line of code we perform 5 instructions:
                

                
    
                    
  1. create a new variable result
    2. 
                    
  2. read num1
    3. 
                    
  3. read num2
    4. 
                    
  4. sum num1's value with num2's value
    5. 
                    
  5. assign the result of the previous expression to result
    6.

Investigating the type of a variable

                

                    When dealing with variables should be really useful to investigate the type of a given variable name.
                

                

                    

                        Does it contain a string? Or a number?
                    

                

                

                    Scenario: you need to sum 2 variables but you want to be sure that thei contains
                    real numbers:
                

                
var a = '1';
console.log(typeof a); // -> string

var b = 1;
console.log(typeof b); // -> number
