Data Types JS Spells

The basic building blocks of a programming language are data types.

You do math operations with numbers, you can validate a mail addess string, you take decision with a true/false value.

As Javascript is a formal language you need to master the syntax that you need to use to work with data types.

Strings

Strings are the most used data type in a normal Javascript script.
We use strings to show messages, and to retrieve some data input.

A Javascript string is a list of characters, punctuation and numbers delimited by an apix like "'": 

'this is a string';

the ";" character after the closing delimiter is an instruction delimiter and Javascript uses it to explicitly divide different instructions.

In point of fact just to declare a string is a perfect instruction in Javascript so it is good practice to put the ";" at the end.

                    The most common Javascript operation is string concatenation with which you can 
                    join two or more strings to become one single data unit.
                

                
'hello' + ' ' + 'world';

                

                    The code above creates 3 strings and uses the "+" operator to concatenate them.
                

                

                    The outcome of such operation is:
                

                
'hello world'

Numbers

                

                    In Javascript you can use numbers and mathematical expressions to do calculations:
                

                
// simple operations:
console.log('1 + 1 =', 1 + 1);
console.log('2 * 2 =', 1 * 1);
                
// Take a google at "MDN Math"
console.log('floor of 4.66 =', Math.floor(4.66));               
console.log('round of 4.66 =', Math.round(4.66));


                

                    

                        Even if you are yet to fully understand what an object and a funtion is you can 
                        try to make a good use of the Math library which provides a lot of extremely useful utilities!
                    

                

                
                

                    Numbers in Javascript are coded as they are, without delimiters. 
                    If a number has decimals then you use a single dot "." as decimal delimiter.
                

                

                    A number which has no decimal part is called integer, a number which has a decimal part is
                    called a float:
                

                
123; // this is an INTEGER number
123.456; // this is a FLOAT number
               
'123'; // this is not a number! this is a STRING!
'123.456'; // this is not a number either! just another string!

Arrays

                

                    An array is a data structure that contains an ordered list of items.
                

                
['blue', 'red', 'green'];

                

                    Arrays are objects just like strings so you can use properties and methods
                    to investigate or modify the content of an array:
                

                
console.log(['blue', 'red', 'green'].length);

                

                    Each item of an array is given an index which can be used to refer 
                    to the item itself later on. 
                


// array in javascript
['blue', 'red', 'green']

// how indexes ar assigned
[0] => 'blue'
[1] => 'red'
[2] => 'green'


                

                    

                        Indexes starts from 0 (zero).

                        The index of the last item is alwasy (arr.length - 1).
                    

                

                

                    Each array item can contain every data types, numbers, strings, etc.
                

                

                    Mozilla Developer Network (MDN) provides an exaustive documentations of the Array object
                    in Javascript:
                

                

                    
                    https://developer.mozilla.org/en/docs/Web/JavaScript/Reference/Global_Objects/Array
